## Description / توضیحات

**English:**
UniversalDB-Java is a comprehensive multi-database learning project built with Java. It demonstrates how to work with different database technologies — including JDBC, Hibernate ORM, JPA, SQLite, PostgreSQL, MySQL, and MongoDB — all inside a single unified structure.

This project is designed for learning, experimentation, and showcasing various data-access patterns in Java. Each module shows a different approach to connecting, querying, and managing data, helping developers understand both SQL and NoSQL systems in a practical way.

**فارسی:**
**UniversalDB-Java** یک پروژه آموزشی و چند‌دیتابیسی با زبان جاوا است که برای یادگیری عمیق نحوه کار با دیتابیس‌های مختلف طراحی شده. در این پروژه، اتصال و کار با انواع تکنولوژی‌ها از جمله **JDBC، Hibernate ORM، JPA، SQLite، PostgreSQL، MySQL و MongoDB** به شکل ماژول‌های جداگانه پیاده‌سازی شده است.

هدف این پروژه، آموزش و نمایش روش‌های مختلف دسترسی به داده در جاوا است. هر ماژول یک روش متفاوت برای اتصال، اجرا کردن کوئری‌ها، مدیریت داده‌ها و درک تفاوت دیتابیس‌های **SQL** و **NoSQL** را ارائه می‌دهد.

## Features / ویژگی‌ها

* Multi-database support (JDBC, Hibernate, JPA, SQLite, PostgreSQL, MySQL, MongoDB)

* SQL and NoSQL demonstration

* CRUD operations for a Library Management System example

* Separate modules for each technology

* Learning-oriented and easy to extend

* پشتیبانی از چند دیتابیس (JDBC، Hibernate، JPA، SQLite، PostgreSQL، MySQL، MongoDB)

* نمایش SQL و NoSQL

* عملیات CRUD برای مثال سیستم مدیریت کتابخانه

* ماژول‌های جداگانه برای هر تکنولوژی

* مناسب برای یادگیری و توسعه آسان

## Project Structure / ساختار پروژه

```
UniversalDB-Java/
 │
 ├── jdbc-mysql/          # JDBC module with MySQL
 ├── hibernate-postgres/  # Hibernate ORM module with PostgreSQL
 ├── jpa-h2/              # JPA module with H2 in-memory database
 ├── mongo/               # MongoDB module
 ├── sqlite/              # SQLite module
 └── README.md            # This file
```

## Getting Started / شروع سریع

**English:**

1. Clone the repository:

```bash
git clone https://github.com/yourusername/UniversalDB-Java.git
```

2. Navigate to the desired module folder and follow instructions in its README.
3. Configure your local database (if required).
4. Build and run the Java project.

1. مخزن را کلون کنید:
